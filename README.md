## Project Overview
Schedule Connect is a web application designed to help UCLA students meet new people and coordinate hangouts by comparing class schedules and identifying shared free time.

Many students struggle to make friends or plan meetups due to busy and mismatched schedules. Our app allows users to upload their schedules, follow other students, compare availability, and message one another based on overlapping free time.

---

## Target Users
- UCLA students
- Students looking to meet peers in their classes
- Friends who want to compare schedules to plan hangouts more easily

---

## App Type
**Social / Communication Web Application**

---

## Core Features
### Required Features
- **Dynamic data**
- **User authentication** (signup/login/logout)
- **Upload + database storage** (schedules, profiles, messages)
- **Search** (find users)

### Additional Features
- **Schedule Upload & Comparison**
  - Users can upload their class schedules
  - Compare schedules with followed users
  - View overlapping classes and shared free time

- **Messaging**
  - Direct messaging between users
  - Helps students connect based on availability

- **Following System**
  - Follow other users
  - Enables schedule comparison and messaging access
